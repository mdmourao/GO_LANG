# Go

### Commands
1. go fmt - format the code
2. go fmt ./... (format all files)
3. go run main.go
4. go build (create exe)
5. go install *.go 
6. go list -m all


### Workspaces 
 
bin - store the compiled files  
pkg - archived (packages storage)  
src - code  

### Packages

```
go get $name
```

### Environment Workspaces
1. GOPATH - my workspace
2. GOROOT - root go
